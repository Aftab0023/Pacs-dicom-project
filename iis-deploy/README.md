# PACS IIS Deployment Package

## Folder Structure After Running deploy.ps1

```
iis-deploy/
├── deploy.ps1              ← Run this as Administrator to deploy everything
├── README.md               ← This file
├── api/                    ← Built .NET API (auto-generated by deploy.ps1)
│   ├── PACS.API.dll
│   ├── appsettings.json    ← Edit DB/Orthanc connection here
│   ├── web.config
│   └── ...
└── frontend/               ← Built React app (auto-generated by deploy.ps1)
    ├── index.html
    ├── config.js           ← Edit API URL here after deployment
    └── ...
```

## Quick Start

1. Open PowerShell as **Administrator**
2. Run: `.\deploy.ps1`
3. Edit `C:\inetpub\pacs-api\appsettings.json` — set your DB connection string
4. Edit `C:\inetpub\pacs-frontend\config.js` — set your server IP
5. Browse to `http://YOUR_SERVER_IP`
